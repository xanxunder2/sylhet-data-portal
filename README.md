# 🌆 SYLHET REGION DATA ELITE (V10)

একটি আধুনিক এবং সুরক্ষিত ড্যাশবোর্ড যা টেলিকম অ্যাক্টিভেশন এবং ডাটা ম্যানেজমেন্টের জন্য ডিজাইন করা হয়েছে। এটি মূলত **Sylhet North Axiata** এর ফিল্ড অপারেশনের জন্য তৈরি একটি লাইটওয়েট ওয়েব টুল।

## ✨ মূল ফিচারসমূহ:
- **🔐 PIN-Protected Security:** নির্দিষ্ট অ্যাক্সেস কোড ছাড়া পোর্টাল ব্যবহার করা সম্ভব নয়।
- **⚡ Automated Data Generation:** জেন্ডার-ভিত্তিক র‍্যান্ডম কাস্টমার প্রোফাইল (নাম এবং নম্বর) তৈরি।
- **📋 Smart Clipboard:** এক ক্লিকেই নাম, নম্বর বা DOBS কোড কপি করার সুবিধা।
- **💾 Local History Log:** আপনার ব্রাউজারে ৫০০টি পর্যন্ত এন্ট্রি সেভ থাকে (LocalStorage)।
- **📥 Data Export:** সংগৃহীত ডাটা সরাসরি `.txt` ফাইল হিসেবে ডাউনলোড করার অপশন।
- **🎨 Ultra-Modern UI:** Tailwind CSS এবং গ্লাস-মরফিজম (Glass-morphism) ইফেক্টসহ রেসপনসিভ ডিজাইন।

## 🛠 প্রযুক্তি (Tech Stack):
- **Frontend:** HTML5, CSS3 (Custom Animations)
- **Styling:** Tailwind CSS (via CDN)
- **Interactions:** Vanilla JavaScript
- **Icons:** Unicode/Emoji based

## 🚀 কিভাবে ব্যবহার করবেন?
১. রিপোজিটরি থেকে `index.html` ফাইলটি ডাউনলোড করুন।
২. যেকোনো ব্রাউজারে ফাইলটি ওপেন করুন।
৩. পিন কোড দিয়ে অথেন্টিকেট করুন (যেমন: `3366` বা `8800`)।
৪. ব্যাচ কোয়ান্টিটি এবং জেন্ডার সিলেক্ট করে 'Start Boom' বাটনে ক্লিক করুন।

## ⚠️ সতর্কতা
এই টুলটি শুধুমাত্র ডাটা ফরম্যাটিং এবং ইন্টারনাল রিপোর্টিং সহজ করার জন্য তৈরি। এতে কোনো রিয়েল-টাইম ডাটাবেস (যেমন Firebase/MySQL) যুক্ত নেই, সব ডাটা আপনার ব্রাউজারের `LocalStorage`-এ জমা থাকে।
